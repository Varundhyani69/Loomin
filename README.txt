# ✨ Loomin - A Modern Social Media Web App

Loomin is a full-stack social media platform built with **MERN Stack** (MongoDB, Express.js, React.js, Node.js) that offers a real-time, interactive user experience with messaging, commenting, sharing, bookmarking, and notification features. It's crafted to deliver an experience similar to Instagram, with a dark modern UI and real-time Socket.io messaging.

---

## 🔥 Features

- 🔐 **User Authentication** – Signup/Login using JWT-based secure auth
- 🖼️ **Post Uploads** – Upload posts with captions and images using `multer`
- ❤️ **Likes & Comments** – Like/dislike and comment on posts with live updates
- 📬 **Real-Time Chat** – Send and receive DMs using `Socket.io` with instant delivery
- 🔖 **Bookmark Posts** – Save your favorite posts
- 🧾 **Edit & Delete** – Edit captions or delete your own posts
- 📢 **Notifications** – Get instant alerts when someone interacts with your post
- 👥 **Follow System** – Follow/unfollow users to curate your feed
- 📲 **Mobile-Responsive Design** – Sleek UI built with Radix UI, TailwindCSS & Lucide icons

---

## ⚙️ Tech Stack

**Frontend**
- React.js
- Redux Toolkit
- Radix UI & TailwindCSS
- Axios
- React Router DOM

**Backend**
- Node.js
- Express.js
- MongoDB + Mongoose
- Socket.io
- Multer (for image uploads)
- JSON Web Tokens (JWT)

**Deployment**
- Deployed on https://loomin-production.up.railway.app/
